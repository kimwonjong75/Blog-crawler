블로그 크롤러 문서

개요

이 프로젝트는 네이버 블로그 글을 수집하고, 선택한 기간의 데이터를 AI로 분석하는 도구입니다. UI는 Streamlit으로 제공되며 로컬 실행을 권장합니다.

실행

pip install -r requirements.txt

python -m streamlit run app.py

환경변수

AI 분석 기능 사용 시 GEMINI_API_KEY가 필요합니다. 운영 환경에서는 비밀키를 저장소에 커밋하지 말고, 제공 플랫폼의 시크릿 스토어를 사용하세요.

자주 묻는 질문

  • 배포 페이지가 404면 index.html 존재 여부와 vercel.json 라우팅을 확인하세요.
  • 콘솔에 확장 프로그램 관련 에러가 보이면 브라우저 확장(예: 검색 레이어)을 일시적으로 비활성화 후 재검증하세요.
